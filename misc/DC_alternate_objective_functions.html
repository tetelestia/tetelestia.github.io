Summary: Alternative objective functions for deep clustering

Synopsis

Deep clustering is good, however with some caveats

  • DC classic optimizes on clustering, and suffers from an unwanted normalization term
  • DC++ optimizes directly on the resultant spectrogram magnitude, however it is an inefficient, unruly beast
  • Noisy phase is used for signal reconstruction

Proposed solution

  • Add new objective functions to deal with undesirable normalization towards clusters of equal size 1
    • Best performing appears to be a Whitened k-means objective function 2
    • All proposed DC objective functions are essentially based on DC's affinity k-means loss function
  • Apply low energy mask to DC objective function
    • Best performing is WMRW_{MR}WMR​

    • We use here magnitude ration weights WMRW_{MR}WMR​ defined as the ratio of the mixture magnitude at T-F bin iii over the sum of the mixture magnitudes at all bins within an utterance: wi=∣xi∣/∑j∣xj∣w_i = |x_i|/\sum_j |x_j|wi​=∣xi​∣/∑j​∣xj​∣, where ∣x∣|x|∣x∣ is the magnitude of the mixture.

  • New Chimera++ network, based on earlier MERL work3, which consists of a two outputs from the BLSTM layer. First, the N×DN\times DN×D DC classic output, second an N×CN\times CN×C output.
    • DC classic output, vt,fv_{t,f}vt,f​ utilizes new objective functions described above
    • New output, M^t,f\hat{M}_{t,f}M^t,f​ utilizes MSE-style, minimum-of-permutations objective function, including phase information 4
      • Called phase-sensitive spectrum approximation (or PSA), using Tab(x)=min(max(x,a),b)T_a^b(x) = min(max(x,a),b)Tab​(x)=min(max(x,a),b) as truncated PSA
    • At inference, only M^t,f\hat{M}_{t,f}M^t,f​ output is required and directly produces spectrogram mask
    • Lchiα++=αLDC(V,Y)+(1−α)LMI\mathcal{L}_{chi_{\alpha}++} = \alpha\mathcal{L}_{DC}(V,Y) + (1-\alpha)\mathcal{L}_{MI}Lchiα​++​=αLDC​(V,Y)+(1−α)LMI​

    • Since we only need to use the MI output at run time, the system is also faster as it does not need to produce embeddings and perform run-time clustering anymore.

  • At inference, iterative method is used for phase reconstruction
    • MISI performed best, applying constraint that reconstructed time-domain signals after each iteration is equal
    • 5 iteations for either MISI or Griffin-Lim used

Results

A new (possibly public?) dataset appears to be used for validation, so previously reported SDR metrics are invalid. In addition, values for CSC (speakers included in training set) and OSC (speakers not seen in training) are reported. All of my results printed here will only be OSC.

Results from previous papers:

  • DC classic = 9.5 dB
  • DC++ = 10.8 dB

Summary of new results:

  • Original objective function:
    • w/ silence masking = 10.2 dB
    • w/ silence mask + MISI = 10.7 dB
  • Best objective function:
    • w/ silence masking = 10.4 dB
    • w/ silence mask + MISI = 10.9 dB
  • Chimera++
    • w/ silence masking = 11.2 dB
    • w/ silence mask + MISI = 11.5 dB

Training notes

  • 8 kHz, 0-10dB attenuation, as per our current training
  • Four BLSTM layers each with 600 units in each direction
  • Entire network trained from scratch on 400-frame segments
  • Trained w/ Adam optimizer, no curriculum LR
  • 0.3 dropout on each BLSTM layer except final
  • No recurrent dropout
  • β\betaβ in WVAW_{VA}WVA​ empirically set to -40dB
  • Only MI loss is used for model selection validation
  • α\alphaα is set through cross-validation
  • FFT appears to be the same as we are using
    • sqrt(Hann), 256-point, 32ms, 8ms
  • Implied near the beginning of \S 5.3, recurrent dropout and curriculum learning may improve results

Speculations

  • DC++ has been phased out in favor of Chimera++ architecture
  • Chimera++ in Alternate Objective Functions paper was not trained as well as possible. Is it possible that its improvement was artificially restricted to increase the apparent benefits of the Unfolded Phase Reconstruction paper? How much benefit does each then provide?
